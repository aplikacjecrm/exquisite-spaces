# Exquisite Spaces – Strona firmowa

Profesjonalna strona firmy Exquisite Spaces Sp. z o.o. z Żywca.

## Uruchomienie

```bash
npm install
npm run dev
```

Otwórz http://localhost:3000

## Build statyczny

```bash
npm run build
```

Wynik w folderze `out/` – gotowy do wgrania na dowolny hosting.

## Zdjęcia

## Technologia

- Next.js 14 (static export)
- TailwindCSS
- Lucide React (ikony)
